# StageHex Trial

Plano gratuito para avaliação completa das ferramentas no SketchUp, sem exportações ou bibliotecas cloud.

***

## StageHex Cloud

| Biblioteca | Disponível |
|------------|------------|
| MA2 Library | ❌ |
| Depence Library | ❌ |

***

## Plugin SketchUp

| Exportação | Disponível |
|------------|------------|
| Exportação MA2 | ❌ |
| Exportação MVR | ❌ |

***

## Ferramentas Disponíveis

| Categoria | Disponível |
|-----------|------------|
| **Ferramentas Core** (interface, mover, rotacionar, organizar, tags, visualização e cores) | ✅ |
| **Event Tools** (plataformas, escadas, guarda-corpos) | ✅ |
| **Rigging Tools** (treliça, tubulação, talhas, cintas e andaimes) | ✅ |
| **Lighting Tools** (fixtures, distribuição, patch completo e modos) | ✅ |
| **Video / Audio Tools** (painéis LED e áudio) | ✅ |
| Distribuição automática de objetos | ✅ |
| Gerenciador de etiquetas | ✅ |
| Patch automático | ✅ |

***

## Ideal Para

* Avaliação das ferramentas StageHex
* Aprendizado e testes de funcionalidades
* Projetos de estudo sem necessidade de exportação

{% hint style="info" %}
O StageHex Trial oferece acesso completo às ferramentas do plugin, permitindo avaliar todo o potencial antes de escolher um plano pago.
{% endhint %}

{% hint style="warning" %}
Este plano não inclui exportações para grandMA2 ou MVR, nem acesso às bibliotecas cloud de fixtures.
{% endhint %}
