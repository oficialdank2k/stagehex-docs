# Rotacionar Objetos

Ferramenta de rotação com precisão para componentes StageHex.

<figure><img src="../../../../.gitbook/assets/gif/main-tools/rotate-tool.gif" alt="Rotacionar Objetos"><figcaption><p>Ferramenta Rotacionar Objetos em ação</p></figcaption></figure>

***

## Como Usar

### Modo 1: Ativar e Selecionar

1. Ative a ferramenta na barra Main Tools
2. Clique nos componentes para selecioná-los
3. Rotacione os componentes para o ângulo desejado

### Modo 2: Selecionar Antes

1. Selecione os componentes desejados
2. Ative a ferramenta
3. Rotacione os componentes

***
