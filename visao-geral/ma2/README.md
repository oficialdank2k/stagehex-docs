# grandMA2

Integração do StageHex com o ecossistema grandMA2, incluindo exportação de fixtures e cena 3D para MA 3D.

***

## Biblioteca StageHex para MA2

A **StageHex Cloud** instala e atualiza automaticamente a biblioteca de fixtures **StageHex** no grandMA2.

{% hint style="info" %}
Basta manter a StageHex Cloud aberta para receber atualizações automáticas da biblioteca.
{% endhint %}
