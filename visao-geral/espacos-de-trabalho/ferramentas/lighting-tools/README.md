# Lighting Tools

Ferramentas para posicionamento, configuração e patch de equipamentos de iluminação.

***

## Inserir e Distribuir

<table>
<thead>
<tr>
<th width="200">Ferramenta</th>
<th>Descrição</th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>Inserir Equipamento</strong></td>
<td>Posiciona fixtures individualmente com orientação automática</td>
</tr>
<tr>
<td><strong>Distribuir Equipamento</strong></td>
<td>Distribui fixtures ao longo de uma linha com espaçamento uniforme</td>
</tr>
<tr>
<td><strong>Distribuir no Truss</strong></td>
<td>Distribui fixtures em faces de truss com clamps e configuração avançada</td>
</tr>
<tr>
<td><strong>Alinhar Equipamento</strong></td>
<td>Alinha e redistribui fixtures selecionados uniformemente</td>
</tr>
<tr>
<td><strong>Configurações</strong></td>
<td>Ajusta Pan/Tilt e clamps dos fixtures selecionados</td>
</tr>
</tbody>
</table>

## Atalhos Comuns

<table>
<thead>
<tr>
<th width="150">Tecla</th>
<th>Ação</th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>Ctrl</strong></td>
<td>Modo Pick - seleciona fixture existente como referência</td>
</tr>
<tr>
<td><strong>Alt</strong></td>
<td>Alternar opções de rotação</td>
</tr>
<tr>
<td><strong>Tab</strong></td>
<td>Mostrar/ocultar fixtures já patcheados (nas ferramentas de Patch)</td>
</tr>
<tr>
<td><strong>Enter</strong></td>
<td>Confirmar e aplicar</td>
</tr>
<tr>
<td><strong>Esc</strong></td>
<td>Cancelar / Sair</td>
</tr>
</tbody>
</table>

{% hint style="info" %}
Todas as ferramentas de patch mostram preview em tempo real antes de aplicar as alterações.
{% endhint %}
