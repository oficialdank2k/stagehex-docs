## Detalhes das Ferramentas

### Painéis LED

#### Inserir Painel LED

Insere um painel LED individual no modelo:

* Seleção de modelo de painel
* Posicionamento preciso
* Configuração de resolução

#### Desenhar Painel LED

Desenha uma tela LED interativamente:

1. Clique no primeiro ponto
2. Arraste para definir a área
3. A tela é criada automaticamente com painéis

### Áudio

#### Inserir Caixa de Som

Insere uma caixa de som individual:

* Seleção de modelo
* Posicionamento e orientação

#### Inserir Arranjo de Áudio

Cria um line array de caixas de som:

* Abre o painel de configuração
* Configuração de quantidade de caixas
* Ângulos entre caixas
* Altura e posicionamento
