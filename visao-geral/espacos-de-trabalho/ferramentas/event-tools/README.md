## Alturas Disponíveis

O sistema suporta diferentes alturas de plataforma:

- **20cm** - Palcos baixos
- **40cm** - Altura padrão
- **60cm** - Palcos elevados
- **80cm** - Estruturas altas
- **100cm** - Plataformas especiais

{% hint style="info" %}
Use as teclas **↑** e **↓** para alternar entre as alturas disponíveis durante a criação.
{% endhint %}
