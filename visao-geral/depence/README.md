# Depence

Integração da StageHex com o Depence através do formato MVR (My Virtual Rig).

***

## Biblioteca StageHex para Depence

A **StageHex Cloud** instala e atualiza automaticamente a biblioteca de fixtures **StageHex** no Depence.

{% hint style="info" %}
Basta manter a StageHex Cloud aberta para receber atualizações automáticas da biblioteca.
{% endhint %}
