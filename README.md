# Stagehex Documentação

StageHex é um plugin profissional para SketchUp voltado ao design técnico de shows, iluminação e estruturas de palco.\
Ele integra modelagem 3D, organização técnica e exportação de dados em um único fluxo de trabalho, permitindo a criação de projetos complexos com precisão e consistência.

O StageHex fornece uma biblioteca estruturada de aparelhos de iluminação, treliças (truss), palcos e elementos cênicos, todos otimizados para uso técnico e organizados por metadados. Isso acelera o desenvolvimento de projetos, reduz erros e garante padronização em produções de qualquer escala.

Um dos principais diferenciais do StageHex é a exportação direta para o formato **MVR (My Virtual Rig)**, assegurando compatibilidade total com softwares de visualização e controle como **Depence²**, **grandMA2** e **grandMA3**. Dessa forma, o projeto desenvolvido no SketchUp pode ser utilizado diretamente na pré-visualização e na programação do show, preservando estrutura, patch, posições e dados técnicos.

Ao transformar o SketchUp em uma plataforma integrada para design técnico, visualização e interoperabilidade, o StageHex elimina retrabalho e aproxima o processo de criação da realidade operacional do show.
