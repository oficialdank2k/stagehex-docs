## Ideal Para

* Profissionais que trabalham com grandMA2
* Projetos que necessitam exportação de cenas para mesa de luz
* Produção de eventos com necessidade de biblioteca de fixtures MA2

{% hint style="info" %}
O StageHex Basic é perfeito para quem precisa exportar projetos para o grandMA2, com acesso à biblioteca completa de fixtures.
{% endhint %}

{% hint style="warning" %}
Este plano não inclui ferramentas avançadas de rigging, distribuição automática ou exportação MVR.
{% endhint %}
