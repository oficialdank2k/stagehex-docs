# Atualização

Atualizações de software, plugins e bibliotecas de assets só podem ser detectadas e aplicadas enquanto o computador estiver conectado à internet.

**Quando uma atualização estiver disponível, a StageHex Cloud notificará o usuário diretamente dentro da aplicação.**\
As notificações são exibidas no canto superior direito da interface do StageHex Cloud.



Basta clicar na atualização desejada e seguir as instruções exibidas na tela.

As atualizações podem incluir:

* Plugin StageHex para SketchUp
* Bibliotecas de Assets
* Integrações com Depence Library
* Conteúdo da MA2 Library
